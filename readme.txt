Organize your Gravity Forms and GravityView forms into folders for better management and organization.

== Description ==

Folders4Gravity provides an intuitive way to organize your Gravity Forms and Gravity Views into folders, making it easier to manage large numbers of forms and views.

= Features =

* **Form Organization**: Create folders to categorize and organize your Gravity Forms
* **View Organization**: Create folders to categorize and organize your GravityView views
* **Intuitive Interface**: Easy-to-use interface for creating, renaming, and deleting folders
* **Form Management**: Duplicate, trash, or edit forms directly from the folder view
* **View Management**: Clone, trash, or edit views directly from the folder view
* **Shortcode Access**: Easily copy form and view shortcodes from the folder view

= Requirements =

* WordPress 5.0 or higher
* PHP 8.0 or higher
* Gravity Forms 2.5 or higher
* GravityView 2.0 or higher (for View Folders functionality)

= Privacy =

This plugin does not collect or store any personal data.

== Installation ==

1. Upload the `folders4gravity` zip file from the Upload Plugin section within the plugin page on the admin dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Forms' > 'Form Folders' to manage your Gravity Forms folders
4. If GravityView is installed, navigate to 'GravityKit' > 'View Folders' to manage your GravityView folders

== Frequently Asked Questions ==

= Can I assign a form or view to multiple folders? =

No, currently a form or view can only be assigned to one folder at a time.

= Will this plugin affect my existing forms or views? =

No, this plugin only adds organizational capabilities and does not modify your existing forms or views.

= What happens to my folders if I deactivate the plugin? =

If you deactivate the plugin, your folders will no longer be visible, but they will remain in the database. If you reactivate the plugin, your folders will be restored.

= What happens to my folders if I uninstall the plugin? =

If you uninstall the plugin, all folder data will be removed from the database.
